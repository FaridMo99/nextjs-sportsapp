# HoopTracker

**HoopTracker** is a NBA Application that shows Basketball Statistics through mainly SSG generated Pages with NextJs

---

## Features

- AI Chatbot
- Searchbar

---

## Tech Stack

- TypeScript
- NextJs
- TailwindCSS
- ShadCN

---

## API's

- [OpenAI API](https://openai.com/api)
- [SportsData.io API](https://sportsdata.io)
